# Cita Previa Extranjería Monitor

Railway-ready monitor for Cita Previa appointment availability with Telegram alerts.

## Files
- `main.py` — monitor logic
- `config.py` — reads environment variables
- `requirements.txt` — Python dependencies
- `Dockerfile` — installs Chromium and ChromeDriver reliably on Railway
- `.env.example` — variables to copy into Railway

## Railway
Create a new GitHub repository, upload all files, create a new Railway project from the repository, and add variables from `.env.example` in Railway Variables.
